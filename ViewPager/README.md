### 간단 기능 ###
> ViewPager와 auto Slide 구현  


### 기능 설명 ###
> + androidx.viewpager.widget.ViewPager 사용  
> + ViewPager Adapter
>   + 5개의 페이지 생성
> + page.xml
>   + 페이지의 밑바탕
> + handler, thread
>   + thread를 이용해 2초마다 화면전환을 명령
>   + 명령을 받으면 handler가 ui를 변경한다.
> + Button
>   + 페이지를 2페이지로
